Create an account

If this is your first time with Framelink, please register yourself introducing your email and choosing a password (you may change it at any time).

In the login box, enable 'Remember me', to save your credentials in the device, for a direct entry in your next access. Disable 'Remember me' to force Framelink to ask email/password every time you try to access it.

Import a Spreadsheet

Click FrameLink icon in the upper left corner and then icon (settings) will appear below FrameLink icon and after the application name.

Select , and choose 'import Spreadsheet' from the pulldown menu.

Framelink will load the data from your Spreadsheet using the spreadsheet content to decide the titles and types for every column.

Video chat with any team member

Click the phone icon to show up all the members of your team. Select the member you want to speak to and imediately you will enter in a video-conference with that member.

Rename a table

Click the table name and replace it by the name you want. The new name will be used instead of the old name, all over FrameLink.

Rename a column

Click the column name and replace it by the name you want. The new name will be used instead of the old name, all over FrameLink.

Send Emails

You can send Emails from any Framelink grid, whenever at least one email path exists. FrameLink will automatically detect any columns of type Email from the current table or any of the related tables in order to establish all possible paths to Email fields.

The above rule applies to all table rows, or only the rows resulting from a prepared query (presets).

If icon is available below the table name, you can send Emails.

To send Emails:

  • Select the rows you want, or select all rows and exclude the rows that you do not want;
  • Click icon and choose the template you want;

Send SMSs

You can send SMSs from any Framelink grid, whenever at least one 'Phone' path exists. FrameLink will automatically detect any columns of type 'Phone' from the current table or any of the related tables in order to establish all possible paths to 'Phone' fields.

The above rule applies to all table rows, or only the rows resulting from a prepared query (presets).

If icon is available below the table name, you can send SMSs.

To send SMSs:

  • Select the rows you want, or select all rows and exclude the rows that you do not want;
